# ProyectoHospitales

# Setup

Descarga Node

Por lo menos **NODE version 8.2.1**

I suggest you install the following that I use in the video:

VSCode: https://code.visualstudio.com/

ITerm2: https://www.iterm2.com/ (Mac Only)

Git: https://git-scm.com/

### Client - Terminal A
```
cd client
npm install
npm start
```

### Server - Terminal B
```
cd server
npm install
npm start
```
